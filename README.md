# NightScreen — сайт кинотеатра

##  Тема проекта
Многостраничный сайт кинотеатра под открытым небом.

##  Описание проекта
Проект представляет собой мини-сайт кинотеатра NightScreen, разработанный с использованием  
HTML, CSS и JavaScript без backend.

Сайт демонстрирует работу с DOM, событиями, пользовательскими функциями,
условиями, циклами, а также валидацию формы.

Проект выполнен индивидуально в рамках итоговой работы по разделу
HTML / CSS / JavaScript.

## Страницы сайта
- **Главная** (`index.html`) — общая информация и навигация
- **Афиша** (`catalog.html`) — список фильмов и сеансов
- **Еда** (`snacks.html`) — снэки, напитки и бургеры
- **О кинотеатре** (`about.html`) — информация о кинотеатре под открытым небом
- **Контакты** (`contacts.html`) — контактные данные и форма обратной связи

##  JavaScript функционал
В проекте реализованы следующие возможности:

- хранение данных о товарах в массивах объектов (снэки, напитки, бургеры);
- фильтрация товаров по цене (до 1000 ₸);
- обработка событий `click` и `submit`;
- расчёт общей стоимости заказа при нажатии кнопки «Заказать»;
- сброс суммы заказа;
- валидация формы:
  - проверка заполнения полей;
  - проверка корректности email;
  - проверка согласия на обработку данных;
- работа с модальным окном (открытие и закрытие схемы зала);
- изменение контента страницы без перезагрузки.
